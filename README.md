#Todo React App

## Assumptions

- node >= 12.22

## Brief

A TODO app with the following features;

- add/remove todo items
- mark/unmark as done
- filter by status

## Tech Stack

- React (UI Library)
- Zustand (State management)
- Typescript (Language)
- Tailwindcss (Styling)
- Jest (Test runner)
- React Testing Library (Testing)
